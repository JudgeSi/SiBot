# SiBot
A command line client for chat-gpt with some helpful features build on top of it.


## Prerequisits
* rust
* cargo
* make (opt)


## Usage
Clone this repo. 

Get your Chat-GPT API key from: https://platform.openai.com/account/api-keys and copy it to a file located at `~/dev/GPT-KEY`

```
make
```

or 

```
cargo run
```
